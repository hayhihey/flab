# Ride-Hailing App Frontend

## Setup Instructions

1. **Install React**: Ensure you have React installed in your project.
2. **Install Dependencies**: Run `npm install` to install the required packages.
3. **Start the App**: Use `npm start` to run the frontend application.

## Components
- **Rider Component**: Handles rider functionalities.
- **Driver Component**: Manages driver functionalities.
- **Admin Component**: Admin panel for managing users and rides.
